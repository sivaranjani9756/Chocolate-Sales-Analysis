# Chocolate-Sales-Analysis
## Overview

This project analyzes chocolate sales data to evaluate overall business performance using SQL, Python (EDA), and Power BI.
The objective is to identify sales trends, product performance, and regional revenue distribution through structured analysis and interactive dashboards.

## Business Objectives

Analyze total sales and revenue trends

Identify top-performing chocolate products

Evaluate regional sales performance

Track key business KPIs

Support data-driven decision-making

## Tools & Technologies

Power BI – Dashboarding & data visualization

SQL (MySQL) – Data analysis & KPI calculations

Python – Exploratory Data Analysis (EDA)

Pandas, NumPy, Matplotlib, Seaborn

## Analysis Components
### 1. SQL Analysis

Revenue and sales aggregation

Product-wise and region-wise performance

Time-based sales analysis (monthly trends)

KPI calculations using GROUP BY and JOINs

### 2. Exploratory Data Analysis (EDA)

Data cleaning and validation

Sales and revenue distribution analysis

Trend and pattern identification

Data quality checks before visualization

### 3. Power BI Dashboard

KPI cards for total revenue and sales

Product-wise and category-wise analysis

Region-wise sales comparison

Time-based trend analysis

Interactive slicers for dynamic filtering

### Key Insights

Identified best-selling chocolate products driving maximum revenue

Observed clear sales trends over time

Highlighted high-performing and underperforming regions

Enabled quick performance comparison using KPIs

### Outcome

This project delivers a complete end-to-end sales analytics solution, demonstrating skills in SQL analysis, Python-based EDA, and Power BI dashboarding.
The insights can help stakeholders optimize product strategy, regional planning, and overall sales performance.
